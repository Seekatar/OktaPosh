# Get-OktaNextUrl

## SYNOPSIS
Get the list of NextUrls the module currently knows about.

## SYNTAX

```
Get-OktaNextUrl
```

## DESCRIPTION
This is for diagnosing the -Next functionality. You can use this to see the url that will be run using -Next
