# Get-OktaApiToken

## SYNOPSIS
Get the API token set by Set-OktaOption or from environment, or passed in

## SYNTAX

```
Get-OktaApiToken [[-ApiToken] <String>] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-OktaApiToken
```

## PARAMETERS

### -ApiToken
