# Get-OktaReadOnly

## SYNOPSIS
Gets the current read-only setting that prevents calling write operations

## SYNTAX

```
Get-OktaReadOnly
```

## DESCRIPTION


## EXAMPLES

### Example 1
```powershell
PS C:\> Get-OktaReadOnly
```

Gets the current value of read only.

## PARAMETERS

## INPUTS

### None

## OUTPUTS

### System.Object
The read only value
