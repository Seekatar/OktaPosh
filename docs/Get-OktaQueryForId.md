# Get-OktaQueryForId

## SYNOPSIS
Get the QueryForId setting

## SYNTAX

```
Get-OktaQueryForId
```

## DESCRIPTION
If QueryForId is set, by default, any Get-Okta* function with a -Query parameter will use heuristics to detect that the query is an id, and retrieve by id.
